# 🔥 Power: Arena 🚀:https://khanrafiya.github.io/Her_Aura/

An AI-powered hand-tracking browser game where your hands become magical weapons. Use real-time gestures to control Fire, Ice, Lightning, and Arcane powers while defending the arena from incoming elemental sigils.

## ✨ Features

* 🖐️ Real-time hand tracking using MediaPipe
* 🔥 Fire & ❄️ Ice elemental powers
* ⚡ Lightning attack mode
* 🔴🔵 Red & Blue Arcane mode
* 🎮 Wave-based survival gameplay
* 👹 Boss waves every 5 rounds
* ❤️ Health & ⚡ Energy system
* 🏆 Score and Combo system
* 🎨 Dynamic particle effects
* 📹 Webcam-based controls
* 🔊 Built-in sound effects
* 💻 Runs directly in the browser

## 🚀 Getting Started

1. Clone or download this repository.
2. Open `index.html` in your web browser.
3. Allow camera access.
4. Click **ENTER THE ARENA**.
5. Use your hands to defeat incoming elemental sigils.

## 🎮 Controls

| Key   | Power                                     |
| ----- | ----------------------------------------- |
| **1** | 🔥 Fire (Left Hand) & ❄️ Ice (Right Hand) |
| **2** | ⚡ Lightning Mode                          |
| **3** | 🔴 Red & 🔵 Blue Arcane Mode              |

### Hand Gestures

* ✋ Open Hand → Activate elemental power
* 🤏 Pinch → Focus/Beam attack
* 🖐️ Left Hand → Fire / Red
* 🖐️ Right Hand → Ice / Blue
* 🙌 Open Hands → Lightning attacks

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript (Vanilla)
* MediaPipe Hands
* HTML5 Canvas API
* WebRTC Camera API

## 🎯 Gameplay

* Destroy elemental sigils before they reach the arena core.
* Match the correct power to each enemy.
* Build combos to earn higher scores.
* Survive increasingly difficult waves.
* Defeat boss sigils every fifth wave.
